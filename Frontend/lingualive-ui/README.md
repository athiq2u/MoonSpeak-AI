# MoonSpeak AI Frontend

This folder contains the Vite + React frontend for MoonSpeak AI.

## Development

```powershell
npm install
npm run dev
```

## Build

```powershell
npm run build
```

## Notes

- The frontend expects the backend API at `http://localhost:5000`.
- Main app entry: `src/App.jsx`
- Full project setup is documented in the repository root `README.md`.
